# Diabetes Prediction Model 🩺

A Machine Learning-based web application that predicts the likelihood of diabetes using patient health parameters. The project combines data preprocessing, feature scaling, model training, and an interactive Streamlit interface to provide quick and reliable predictions.

## 🚀 Features

- Predicts diabetes risk based on medical parameters.
- Interactive Streamlit web interface.
- Data preprocessing and feature scaling.
- Machine Learning-based classification model.
- Real-time prediction generation.
- Simple and user-friendly design.

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

## 📊 Dataset

The model is trained on a diabetes dataset containing health-related features such as:

- Pregnancies
- Glucose Level
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age

These attributes are used to determine whether an individual is likely to have diabetes.

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/Diabetes-Prediction-Model.git
```

### Navigate to the Project Folder

```bash
cd Diabetes-Prediction-Model
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```
